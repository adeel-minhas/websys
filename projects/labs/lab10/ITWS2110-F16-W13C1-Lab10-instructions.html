Lab #10 - PHP and Postgres

  • Part 0 - Install Postgres & create database
    • Go to PostgreSQL Downloads and select the installation option that fits with your operating system
    • Create a new folder for Lab 10 in your websys folder. At this point, you should have a virtual server pointing to an index.php file in your websys folder being served as a virtual host on your web server.
    • Using Postgres terminal (psql) to create a database named yourRCSid-websyslab10.
  • Part 1 - install.php
    • Using the PHP Postgres API create tables that have the same structure as those in the MySQL database yourRCSid-websyslab9 from Lab 9.
  • Part 2 - index.php
    • Create an index.php to display a webpage which will provide the user with 2 options:
      1. Choice 1, Install will:
        • call install.php and display its output
      2. Choice 2, Migrate will:
        • SELECT all records in all tables from the MySQL database from lab 9
        • INSERT those records into the PostgreSQL database
      3. Choice 3, Select will:
        • Query the PostgreSQL database
        • SELECT all students (first and last name) who have an A(90+) grade in at least one class, and the class (course name) they have the most A's in. You may break ties however you wish
        • Display the data in a table
  • Receiving credit
    • Export the structure and content of your Postgres database and tables to a text file.
    • Display the output of the queries in Choice 3 in the browser.
    • As normal, there should be a readme file included which explains your observations/comments/assumptions from this lab.
